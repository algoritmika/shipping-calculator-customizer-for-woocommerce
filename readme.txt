== Description ==

**Shipping Calculator Customizer for WooCommerce** is a lightweight plugin that lets you customize the shipping calculator on the WooCommerce cart page.

Enhance your WooCommerce cart with the "Shipping Calculator Customizer for WooCommerce" plugin, offering customizable options for a streamlined, user-friendly shipping estimation process that aligns with your brand's style.

== Installation ==

1. Upload the entire plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start by visiting plugin settings at "WooCommerce > Settings > Shipping Calculator Customizer".
